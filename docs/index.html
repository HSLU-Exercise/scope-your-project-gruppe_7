Schnellstart

git clone https://github.com/HSLU-Exercise/scope-your-project-gruppe_7.git
cd scope-your-project-gruppe_7

# Installation (je nach Projektechnologie)
pip install -r requirements.txt   # Python
npm install                      # Node.js

# Server starten
python app.py      # oder
npm run dev
Commit – Lint & Tests laufen automatisch
Build – Release-Artefakte werden versioniert
Deploy – GitHub Pages / Cloud
